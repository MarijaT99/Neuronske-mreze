
# Hijerarhijska klasifikacija biljnih bolesti korišenjem dubokih neuronskih mreža

Master projekat iz predmeta *Neuronske mreže*, FTN, Univerzitet u Novom Sadu.
Autor: Marija Tadić.

Klasifikacija biljnih bolesti sa fotografija listova (dataset **PlantVillage**),
poređenje *flat* klasifikatora (sve klase odjednom) i **hijerarhijskog** pristupa
(prvo vrsta biljke, zatim bolest unutar te vrste).
